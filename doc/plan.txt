board-app (phyWAVE-KW22):
	- liest regelmäßig (ca. einmal pro sekunde) die Werte der Sensoren aus
	  und schreibt die Werte in die LEDs
	- auf Anfrage durch Server werden die aktuellen Sensorwerte gemäß Protokoll
	  (protocoll.txt) an den Server übertragen
	- auf Knopfdruck wird der Alksensor aktiviert und die Werte auf einen
	  LED-Streifen geschrieben

server-emulator (atmel-samr21):
	- auf Knopfdruck werden der nach einander die Daten aller registrierten
	  Boards abgefragt
	- Sensordaten werden auf der stdout ausgegeben
